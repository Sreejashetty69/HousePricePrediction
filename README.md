# House Price Prediction (Machine Learning)

This is a beginner Machine Learning project to predict house prices based on size and number of bedrooms.

## Features
- Loads housing dataset
- Trains Linear Regression model
- Predicts house price for user input
- Displays model accuracy
- Visualizes data using graph

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## What I Learned
- Data loading and preprocessing
- Training a machine learning model
- Model evaluation
- Making predictions
- Data visualization

## How to Run
1. Install required libraries:
   pip install pandas scikit-learn matplotlib
2. Run:
   python house_price.py
